# Documentation

This directory contains setup guides and problem statements for the React interview practice playground.

## Files

### `init-playground-project.md`
Complete setup guide for initializing a Vite + React + TypeScript project with:
- Project initialization steps
- Recommended folder structure
- Enhanced tooling setup (Prettier, ESLint, Tailwind CSS)
- pnpm configuration and usage

### `problem-statements.md`
Curated list of 12 React frontend interview problems organized by category:
- **State Management & Hooks** - Todo list, Debounce, Phone formatter
- **Components & Rendering** - Counter undo/redo, Infinite scroll, Accordion
- **Performance & Optimization** - Memoization, Lazy loading, Context API
- **Integration & Advanced** - Data fetching, Spreadsheet UI, Game of Life

Each problem includes:
- Why it's relevant
- Tips for implementation
- Links to resources (LeetCode, GreatFrontend, etc.)

## Quick Reference

- See `../CLAUDE.md` for development guidance
- See `../src/features/README.md` for how to add new features
- Run `pnpm dev` from the project root to start the development server
